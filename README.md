# PlatinumRx Assignment

This project demonstrates my skills in SQL, Excel, and Python through practical problem-solving scenarios.

---

## 1. SQL Proficiency

### Hotel Management System

* Created tables: users, bookings, booking_commercials, items
* Inserted sample data
* Solved queries including:

  * Last booked room per user
  * Total billing for a given month
  * Bills greater than a threshold
  * Most and least ordered items per month
  * Second highest bill per month

### Clinic Management System

* Created tables: clinics, customer, clinic_sales, expenses
* Inserted realistic data
* Solved queries including:

  * Revenue by sales channel
  * Top 10 customers
  * Monthly revenue, expense, and profit
  * Most profitable clinic per city
  * Second least profitable clinic per state

---

## 2. Spreadsheet Proficiency

* Created two sheets: ticket and feedbacks
* Used XLOOKUP to populate ticket_created_at in feedbacks
* Created helper columns:

  * Same Day → check if created and closed on same date
  * Same Hour → check if created and closed within same hour
* Used COUNTIFS to calculate outlet-wise counts for:

  * Same day tickets
  * Same hour tickets
